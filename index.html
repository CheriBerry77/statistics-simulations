Population Truth ›
—
actual parameter (p)

The population parameter is the true proportion of YES answers in the entire population. In real life this is unknown — we use samples to estimate it.

Formula
p = (# YES in population) / N
In this simulation
p = (Σ YESA + Σ YESB) / N
  • Symbol p (proportion) or μ (mean)
  • Type Parameter — describes the whole population
  • Known? Only in simulations. In practice, we never know p exactly.
Sample Result ›
—
this sample's statistic (p̂)

The sample proportion p̂ ("p-hat") is the fraction of YES responses in your sample. It is our best estimate of the unknown population parameter p.

Formula
p̂ = x / n
where x = number of YES in sample, n = sample size
Standard Error of p̂
SE(p̂) = √[ p̂(1 − p̂) / n ]
How much p̂ varies from sample to sample
  • Symbol p̂ (p-hat)
  • Type Statistic — computed from the sample
  • Larger n → p̂ gets closer to p on average
Estimated Bias ›
—
sample − population

Bias is a systematic error that causes the sample statistic to consistently over- or under-estimate the true parameter. Unlike random variability, bias does not shrink with larger samples.

Formula (single sample)
Bias ≈ p̂ − p
True definition (over many samples)
Bias = E[p̂] − p
E[p̂] = expected value of p̂ across all possible samples
Mean Squared Error (MSE)
MSE = Bias² + Variance(p̂)
Total error = systematic error + random error
  • Unbiased E[p̂] = p → bias = 0
  • Positive bias p̂ > p → overestimates truth
  • Negative bias p̂ < p → underestimates truth
Variability (after 100×) ›
—
std dev of sample proportions

Variability measures how spread out the sample proportions are across many repeated samples. A low-variability method consistently gives similar results; a high-variability method swings wildly.

Theoretical Standard Deviation of p̂
σp̂ = √[ p(1 − p) / n ]
Uses the true p; decreases as n increases
Simulated Std Dev (after 100 runs)
s = √[ Σ(p̂ᵢ − p̄)² / (k − 1) ]
p̄ = mean of all sample proportions, k = number of runs
Central Limit Theorem
p̂ ~ N( p, √[p(1−p)/n] ) when n is large
Requires np ≥ 10 and n(1−p) ≥ 10
  • Larger n → smaller σp̂ → less variability
  • Max variability when p = 0.5 (σp̂ is largest)
  • vs. Bias Variability is random error; bias is systematic error

Core Concepts

🎯

Bias

A consistent, systematic error in which a sample over- or under-represents the population. Bias comes from the method, not bad luck.

Watch for: Convenience and voluntary response sampling almost always produce bias. SRS and stratified sampling minimize it.
📊

Variability

How spread out sample results are across many repetitions. Larger samples produce lower variability — the results cluster near the truth.

Try it: Run 100 simulations with n=20, then n=200. Watch the distribution compress.
🗺

Sampling Distribution

The distribution of all possible sample statistics from a population. The 100× view in this simulator shows you what one looks like in practice.

Key insight: The center of the sampling distribution reveals bias. The spread reveals variability.
🔬

Inference

Using sample data to draw conclusions about the population. Only valid when the sample is representative — which requires a good sampling method.

Remember: A confidence interval is meaningless if the sample is biased.

Sampling Method Comparison

Method How it works Bias risk Practical? Best for
SRS Every individual has an equal chance of selection Low Moderate Gold standard; textbook default
Stratified Divide into groups, SRS within each group Low Moderate When groups differ meaningfully
Cluster Randomly select whole groups (clusters) Medium High Geographically spread populations
Convenience Sample whoever is easiest to reach High Very high Avoid for inference
Voluntary Response People self-select into the sample High Very high Avoid for inference

Glossary

Population (N)
The entire group you want information about.
Sample (n)
The subset of the population actually measured.
Parameter
A number describing the population (e.g. true proportion p).
Statistic
A number computed from the sample (e.g. sample proportion p̂).
Sampling Frame
The list of individuals from which the sample is drawn.
Undercoverage
When some groups are underrepresented in the sampling frame.
Non-response Bias
When individuals who don't respond differ from those who do.
Response Bias
When the question wording or context distorts answers.
Margin of Error
The ± range in a confidence interval: z* × √(p̂(1−p̂)/n).
Central Limit Theorem
For large n, p̂ is approximately Normal regardless of population shape.
